# コンセプト

この言語は絶対的な安全性となめらかな使い心地を意識した言語です。

## 1.強い静的型付け

多くの言語では静的型付けですがこの言語ではさらにハイレベルな強い静的型付けを採用しています。\
強い静的型付けでは演算では常に全く同じ型を要求し、もし合わなければエラーとなります。

黙認的キャストは実装されず、演算量が多い、データ損失やフォーマットによりasキャストですらできない場合があります。
### 1.1.例外的な黙認的キャスト
ただし、リテラル型に限り、黙認的キャストが実装されている

# データ構造

すべてのオブジェクトは型と値を持っています。\
他のオブジェクトに参照を持つような型はジェネリックな型です。

## 整数

### Int系

`Int8`, `Int16`, `Int32`, `Int64`, `Int128`, `UInt8`, `UInt16`, `UInt32`, `UInt64`, `UInt128`, `Char`, `Short`,`Int`, `Long`がサポートされており、\
`Char`は`Int8`,`Short`は`Int16`,`Int`は`Int32`,`Long`は`Int64`と同値です。\
Asキャストが使用できますがデータの損失が発生する可能性があります。

### Num

無制限の整数型です。

Int型と相互的なAsキャストが可能です。データの損失が発生する可能性があります。

## 小数

### Float系

`float32`, `float64`, `float`, `double`をサポートしています。\
`float`は`float32`, `double`は`float64`と同値です。

### Dec

`Dec`は小数点の精度が保証されています。

## Boolean

真偽値型です。

TrueとFalseを持ち、キャストができません。

## 文字列

`String`がサポートされています。
任意の文字列が保存可能です

## None

Noneです。無を意味します

### Null

Noneに含まれる型です。
Nullは無効ポインタを意味し、参照はできません。

## Any

任意のオブジェクトを束縛できます

## Dynamic

型推論型です。

## ジェネリックな型

### List<T>

可変長のオブジェクトを持つコンテナです。

添字表記`a[k]`は、`k`でインデクス指定された要素を`a`から選択します

### Array<T, L>

長さ`L`の有限のオブジェクトを持つコンテナです。

添字表記`a[k]`は、`k`でインデクス指定された要素を`a`から選択します

### Map<K, V>

添字表記`a[k]`は、`k`でインデクス指定された要素を`a`から選択します

### Ptr<T>

生ポインタです。

`T`のアドレスです。

### &borrow<T>

`T`の借用型です。

# Stmt(文)

## 変数宣言

変数に値が束縛されます。\
束縛タイプには次のものがあります。\
`let`, `val`, `const`, `mut`, `borrow`

構文は常に

```
<束縛タイプ> <型> <識別子>;
```
または
```
<束縛タイプ> <型> <識別子> = <初期値>;
```
を要求します。

### let
`let`は可変なC変数です。

### val
`val`は不変だが動的に変更可能なC変数です。

### const
`const`は常に初期値を取り続ける定数変数です。

### mut
`mut`は可変なRust変数です。

### borrow
`borrow`は借用を取る不変なRust変数です。常にmutの借用を要求します。

## 式文

式を持っています。

## ブロック
ブロックはスコープを持ちます。\
ブロックを用いることで、複雑な演算がまとめられます。

## If文
`if文`は条件分岐の為の文です\
条件がTrueならばthenStmtが実行され、\
FalseならばelseStmtがあれば実行されます
```
if <条件式> <thenStmt>
```
または
```
if <条件式> <thenStmt>
else <elseStmt>
```

## While文
`While文`は条件式がTrueならば常に繰り返します。
```
while <条件式> <Stmt>
```

## 関数定義
関数は引数、返り値、文が必要です。
```
fn <型> <識別子>(<引数>) <Stmt>
```
`<引数>`は
```
<型> <識別子>
```
のグループです。

## Return文
関数から帰る際に使用されます。
```
return <Expr>;
```
## For文
```
for <iterator> in <iterable> <Stmt>
```
の構文で実行されます。\
`<iterator>`は繰り返しに入る変数であり、\
`<iterable>`はジェネリックなオブジェクトです。

## import文
```
import <path>;
```
`<path>`を導入します
