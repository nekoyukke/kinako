## 行動規範

すべてのやり取りにおいて、敬意と建設的な姿勢を心がけてください。私たちは、すべての貢献者にとって歓迎され、誰もが参加できる環境を提供することに尽力しています。

## 貢献方法

### 1. バグの報告 🐛

バグを発見しましたか？以下の情報を添えて Issue を作成してください。

**タイトル**: バグの簡単な説明

**説明**:
- 想定していた事象
- 実際に発生した事象
- 再現手順

**環境**:
- pythonのバージョン（py --version, python --version）
- OS (Windows/macOS/Linux)・バージョン
**ラベル**: `bug` ラベルを追加

**例:**
**タイトル** : ネストした関数のパーサでのエラー
**説明**: 以下のファイルを作成し、実行するとエラーが起こる
```
fn int main(){
    fn int foo(){
        return 0;
    }
    return foo;
}
```
